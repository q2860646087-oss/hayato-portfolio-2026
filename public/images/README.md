# 图片目录说明

项目图片统一放在：

```text
public/images/projects/
```

当前 3 个项目分别对应：

```text
public/images/projects/project-01/
public/images/projects/project-02/
public/images/projects/project-03/
```

每个项目目录建议保留以下文件名：

```text
cover.jpg
moodboard.jpg
process-01.jpg
process-02.jpg
final-01.jpg
final-02.jpg
```

替换图片时，直接覆盖同名文件即可。如果你想使用更多图片或改文件名，请同步修改：

```text
data/projects.ts
```
