Load the `react-native` skill.

Parse $ARGUMENTS:
- File path → review the file against React Native patterns
- `setup` → guide through new Expo project setup
- `review` → review the current project's mobile code

Examples: `/react-native src/screens/Home.tsx`, `/react-native setup`, `/react-native review`
