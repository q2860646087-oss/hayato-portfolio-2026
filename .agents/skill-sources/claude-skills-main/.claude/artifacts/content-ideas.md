# Content Ideas

## Teach Patterns, Not Ship Scripts
Designing AI skills that actually work for other users. Why pre-built scripts break (wrong Python version, missing deps, macOS-only paths, interactive prompts in non-interactive shells). The pattern-vs-script tradeoff. How to structure `references/` files. Real example: removing 17 scripts and creating 6 pattern docs. Audience: Claude Code skill authors and AI tool builders.
