# Mobile Portfolio UI — 手机端作品集 UI 重建 Skill

> 本 Skill 专用于 `portfolio-2026_V1_Hero_Final` 项目的手机端适配。
> 核心原则：**只改手机端，桌面端一字不动。**

## 约束（必须遵守）

1. **只改手机端**。所有新增 CSS 必须写在 `@media (max-width: 768px) { ... }` 内部。
2. **桌面端现有效果不允许被影响**。任何不在 `@media (max-width: 768px)` 内的修改都必须拒绝。
3. 触屏交互专属样式使用 `@media (hover: none) and (pointer: coarse) { ... }`。
4. **优先改 `app/globals.css`**。不要在组件文件中添加样式。
5. 如果必须改组件，只允许：
   - 添加 `mobile-only` className
   - 添加极小的条件判断（如 `isTouchOnly`）
   - 不允许改组件结构、布局、props
6. **不允许改**：Canvas、WebGL、3D、图片资源、路由、文案、动画删除。
7. 允许优化：手机端字号、间距、导航、图片比例、内容顺序、滚动动画稳定性。
8. 手机端动画**只优先使用 `opacity` 和 `transform`**。
9. **不要用** `display`、`height`、`width`、`margin`、`top`、`left` 做进入/离开动画。
10. 需要处理：边缘闪烁、滚动渐入渐出抖动、导航贴边抽搐、文字贴边拥挤问题。

## 字号规范（手机端）

| 元素 | 建议范围 |
|---|---|
| Hero 大标题 | 36px–48px |
| 页面主标题 | 28px–36px |
| 模块标题 | 22px–28px |
| 副标题 | 16px–20px |
| 正文 | 15px–17px |
| 注释/标签 | 12px–14px |
| 导航文字 | 13px–16px |

## 安全边距

- 侧边导航距屏幕边缘：16px–24px
- 页面内容左右边距：至少 16px
- Header 距顶部状态栏：至少 8px

## 工作流程

1. **每次修改前先说明计划**：列出要改的文件、要改的 CSS 选择器、手机端生效范围。
2. **修改后汇报**：
   - 修改了哪些文件
   - 哪些代码只在手机端生效（`@media (max-width: 768px)` 内）
   - 桌面端为什么不会受到影响
   - 是否改动了任何桌面端样式或组件逻辑
3. **如果不确定是否影响桌面端，先问再改。**

## 适用场景

- 手机端字号过大/过小
- 手机端元素贴边、拥挤、裁切
- 手机端导航栏 hover 闪烁
- 手机端滚动渐入动画在视口边缘抖动
- 手机端 3D Canvas 比例失调
- 手机端跑马灯卡片尺寸不合适
- 任何需要"只改手机端"的场景
